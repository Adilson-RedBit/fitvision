## ⚡ OPÇÃO C: HTML/JS SIMPLES (Rápido)

### **Stack Minimalista:**

```html
- HTML5
- TailwindCSS (via CDN)
- Alpine.js (reatividade)
- Chart.js (gráficos)
- Vanilla JS
```

### **Estrutura:**

```
frontend-simple/
├── index.html           (Login)
├── dashboard.html       (Dashboard)
├── hotels.html          (Gerenciar hotéis)
├── faqs.html            (Gerenciar FAQs)
├── conversations.html   (Ver conversas)
├── js/
│   ├── api.js          (Chamadas à API)
│   ├── auth.js         (Autenticação)
│   └── utils.js        (Helpers)
└── css/
    └── custom.css      (Estilos extras)
```

### **Exemplo de Dashboard:**

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <title>Chatbot Hoteleiro - Dashboard</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>
<body class="bg-gray-100" x-data="dashboard()" x-init="init()">
    <!-- Sidebar -->
    <aside class="fixed left-0 top-0 h-screen w-64 bg-white shadow-lg">
        <!-- Navigation -->
    </aside>

    <!-- Main Content -->
    <main class="ml-64 p-8">
        <!-- KPIs -->
        <div class="grid grid-cols-4 gap-4 mb-8">
            <div class="bg-white p-6 rounded-lg shadow">
                <h3 class="text-gray-500 text-sm">Total Conversas</h3>
                <p class="text-3xl font-bold" x-text="stats.totalConversations">0</p>
            </div>
            <!-- More cards -->
        </div>

        <!-- Charts -->
        <div class="grid grid-cols-2 gap-4">
            <canvas id="conversationsChart"></canvas>
            <canvas id="resolutionChart"></canvas>
        </div>
    </main>

    <script>
        function dashboard() {
            return {
                stats: {},
                async init() {
                    const response = await fetch('http://localhost:3000/api/metrics/dashboard', {
                        headers: {
                            'Authorization': 'Bearer ' + localStorage.getItem('token')
                        }
                    });
                    const data = await response.json();
                    this.stats = data.data;
                }
            }
        }
    </script>
</body>
</html>
```

### **Tempo de Desenvolvimento:**

- Setup: 30 min
- Login page: 1h
- Dashboard: 2h
- CRUD pages: 3h
- Integração API: 2h
- Polish: 1h
- **TOTAL: 9-10 horas (~2 dias)**

### **Vantagens:**
- ✅ Muito rápido
- ✅ Zero build process
- ✅ Fácil de entender
- ✅ Sem dependências npm
- ✅ Deploy simples

### **Desvantagens:**
- ❌ Menos profissional
- ❌ Difícil de escalar
- ❌ Menos features
- ❌ SEM roteamento SPA
